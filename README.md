## 🚀 Overview

This repository is a comprehensive guide to the **AI Meeting Notes Ecosystem**. We track the leading **SaaS platforms** and **Open-Source projects** that are revolutionizing how we capture and utilize meeting data.

### 🌟 Key Features of Featured Tools:
- **Transcription**: Real-time or post-meeting speech-to-text.
- **Diarization**: Intelligent speaker identification.
- **Summarization**: Context-aware summaries and action items.
- **Agentic Capabilities**: AI agents that can search history and provide insights.
- **Privacy First**: Tools that support local LLMs (Ollama) and self-hosting.

## 📂 Open-Source GitHub Projects

### Self-Hosted Assistants
*Full control over your data with privacy-centric AI.*

- **[Meetily](https://github.com/Zackriya-Solutions/meetily)** 🏠 — **Top Choice**. Privacy-first, local Whisper transcription & Ollama summarization.
- **[Char (anarlog)](https://github.com/fastrepl/anarlog)** 📝 — Open-source Granola alternative for personalized meeting memos.
- **[Pensieve](https://github.com/lukasbach/pensieve)** 🧠 — Local-only desktop app for private meeting recording and summarization.

### Local Transcription Tools
- **[Ownscribe](https://github.com/paberr/ownscribe)** — CLI-based transcription using WhisperX and Phi-4-mini.
- **[Meetscribe](https://github.com/pretyflaco/meetscribe)** — Local diarization and professional PDF output.
- **[Meeting-Transcriber](https://github.com/pasrom/meeting-transcriber)** — macOS menu bar app for seamless on-device processing.
- **[AI-Powered-Meeting-Summarizer](https://github.com/AlexisBalayre/AI-Powered-Meeting-Summarizer)** — Simple Gradio interface for Whisper + Ollama.
- **[Summeet](https://github.com/lyzgeorge/summeet)** — FastAPI + Vue stack for audio-to-structured-summary.

## 🛠️ Frameworks & APIs

Build your own AI meeting agent using these powerful foundations:
- **[Whisper](https://github.com/openai/whisper)** — The gold standard for open-source speech-to-text.
- **[Ollama](https://ollama.ai/)** — Run powerful LLMs locally for summarization.
- **[LangGraph](https://github.com/langchain-ai/langgraph)** — Orchestrate multi-agent meeting workflows.
- **[Pyannote-audio](https://github.com/pyannote/pyannote-audio)** — Advanced speaker diarization toolkit.
