## 🚀 Overview

This repository is the ultimate guide to the **AI Meeting Notes Ecosystem**. We track the leading **SaaS platforms** and **Open-Source projects** that are revolutionizing how teams capture and utilize meeting data.

### ❓ Why use AI for Meeting Notes? (SEO)
Using **AI meeting assistants** and **automated transcription** tools helps teams:
- **Save Time**: No more manual typing or re-watching 1-hour recordings.
- **Boost Accuracy**: Get near-perfect transcripts with **Whisper-based AI**.
- **Actionable Insights**: Extract **AI-generated action items** and key decisions instantly.
- **Privacy & Security**: Use **local LLMs (Ollama)** to keep sensitive data on your machine.
- **Searchable Knowledge**: Turn your meeting history into a searchable database.

## 📂 Open-Source GitHub Projects

### 🏠 Self-Hosted Assistants
*Full control over your data with privacy-centric AI.*

- **[Meetily](https://github.com/Zackriya-Solutions/meetily)** 🏠 — **Top Choice**. Local Whisper + Ollama. 🍎 🪟 🐧
- **[Char (anarlog)](https://github.com/fastrepl/anarlog)** 📝 — Open-source Granola alternative. 🍎 🪟
- **[Pensieve](https://github.com/lukasbach/pensieve)** 🧠 — Local-only desktop app. 🍎 🪟

### 💻 Local Transcription Tools
- **[Ownscribe](https://github.com/paberr/ownscribe)** ⌨️ — CLI-based using WhisperX. 💻
- **[Meetscribe](https://github.com/pretyflaco/meetscribe)** 📄 — Local diarization & PDF output. 🪟 🐧
- **[Meeting-Transcriber](https://github.com/pasrom/meeting-transcriber)** 🧭 — macOS menu bar app. 🍎
- **[AI-Powered-Meeting-Summarizer](https://github.com/AlexisBalayre/AI-Powered-Meeting-Summarizer)** 🌐 — Gradio interface for Whisper. 🐳
- **[Summeet](https://github.com/lyzgeorge/summeet)** ⚡ — FastAPI + Vue stack. 🐧
